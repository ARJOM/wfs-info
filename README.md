# WFS Info
Repositório destinado ao consumo de dados em WFS, utilizando a biblioteca OWSLib


### Executando
1. Criar ambiente virtual
    ```
    python3 -m venv venv
    ```
2. Ativar ambiente virtual
    ```
    source venv/bin/activate
    ```
   2.1. Atualizar o pip
   ```
    pip install --upgrade pip
    ```
3. Instalar dependências
    ```
    pip install -r requirements.txt
    ```
4. Executar servidor
    ```
    python run.py
    ```
